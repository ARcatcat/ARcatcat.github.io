# 游戏亮点

1. 应用了AR技术在传统养猫游戏上，能基于现实场景产生AR场景，并让猫在AR场景下活动
2. 小猫能进入不同场景活动，并且不同场景之间能够自由切换
3. 在活动的过程中，小猫能够自动规避场景中的障碍物
